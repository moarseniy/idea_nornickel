# IDEA

**IDEA — Intelligent Discovery & Evaluation Assistant** — локальный MVP интерпретируемой ИИ-платформы для генерации, обоснования и ранжирования научно-исследовательских гипотез.

Система принимает KPI проекта и базу знаний: публикации, патенты, отчеты, результаты экспериментов и технологические документы. IDEA извлекает сущности, строит динамический граф знаний, генерирует проверяемые гипотезы и ранжирует их по новизне, риску, реализуемости и потенциальному эффекту.

## Возможности

- загрузка PDF, DOCX, XLSX, CSV, TXT, MD, PNG/JPG;
- мультиязычный разбор источников и OCR;
- динамический интерактивный граф знаний;
- генерация гипотез через OpenAI API;
- web research со ссылками на статьи, патенты и кейсы;
- экономический контур и план проверки/внедрения;
- feedback, чат, статусы, версии;
- экспорт CSV, Markdown и PDF.

## Запуск

```bash
cp .env.example .env
```

Добавьте ключ в `.env`:

```env
OPENAI_API_KEY=...
```

Запустите приложение:

```bash
docker compose up --build
```

Откройте в браузере:

```text
http://localhost:8080
```

Без `OPENAI_API_KEY` можно загружать и разбирать документы, но генерация гипотез, чат и web research работать не будут.

## Данные

- `storage/` хранит SQLite-базу, загруженные файлы и OCR-модели.
- `data/` можно использовать как локальную папку с демо-источниками; она монтируется в контейнер как `/app/data`.
